# Memory directory

Runtime-derived state for the three-doc execution system.

## Files generated here

- `source-manifest.json` — SHA256 hashes of the 3 source docs
- `architecture-contract.md` — extracted architecture from the technical guide
- `decisions.md` — architecture decision log
- `risk-register.md` — identified risks
- `project-brief.md` — one-page project brief
- `active-phase.md` / `active-phase.json` — current phase context
- `active-task.md` / `active-task.json` — current task context
- `official-doc-notes/` — notes from official documentation checks

These are derived artifacts. The source of truth is always `docs/source-of-truth/`.
